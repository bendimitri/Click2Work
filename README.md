# ServiçosJa

Plataforma web responsiva estilo marketplace de serviços, feita apenas com HTML, CSS e JavaScript puro.

## Como rodar

1. Abra `index.html` diretamente no navegador; ou
2. Publique a pasta no GitHub Pages.

Não há dependências, build, back-end ou banco de dados. A aplicação usa dados mockados, sessão local e `localStorage`.

## Acessos demo

- Contratante: `demo@contratante.com` / `123456`
- Trabalhador: use qualquer e-mail mockado de `js/data.js` / `123456`

## Recursos

- Landing page completa
- Login e cadastro fake para contratantes e trabalhadores
- Dashboard por tipo de usuário
- Busca e filtros de profissionais
- Perfis detalhados com avaliações e portfólio
- Favoritos com persistência local
- Chat simulado com histórico salvo
- Assistente ServiçosJa IA simulado
- Edição de perfil e serviços do trabalhador
- Modo claro/escuro
- Layout responsivo para desktop, tablet e celular
